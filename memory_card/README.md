# بازی حافظه کارتی | Memory Card Game

یک بازی حافظه کارتی کاملاً تحت وب با قابلیت‌های پیشرفته و چالش‌برانگیز

## ویژگی‌های اصلی

### قابلیت‌های بازی
- **سه سطح سختی**: آسان (4×4)، متوسط (6×4)، سخت (6×6)
- **سه تم کارت**: ایموجی، اعداد، رنگ‌ها
- **سیستم امتیازدهی هوشمند**: امتیاز بر اساس زمان، حرکات و دقت
- **تایمر پیشرفته**: زمان‌سنج با قابلیت توقف و ادامه
- **شمارنده حرکات**: ثبت تعداد حرکات انجام شده
- **انیمیشن‌های روان**: انیمیشن چرخش کارت‌ها و افکت‌های جذاب
- **افکت‌های صوتی**: صداهای مختلف برای حرکات، تطابق و عدم تطابق

### محاسبه امتیاز
امتیاز بازی بر اساس سه عامل محاسبه می‌شود:
- **پاداش جفت**: 100 امتیاز برای هر جفت یافت شده
- **پاداش زمان**: حداکثر 50 امتیاز اضافی برای سرعت بالا
- **پاداش حرکت**: حداکثر 30 امتیاز اضافی برای کمترین حرکات

### پشتیبانی چند زبانه
- **فارسی**: به‌عنوان زبان پیش‌فرض با طراحی RTL
- **انگلیسی**: با طراحی LTR
- تبدیل خودکار جهت صفحه با تغییر زبان
- ترجمه کامل تمام المان‌های رابط کاربری

### طراحی و رابط کاربری
- **طراحی ریسپانسیو**: سازگار با تمام اندازه‌های صفحه (دسکتاپ، تبلت، موبایل)
- **حالت تاریک/روشن**: تم‌های قابل تعویض با طراحی مدرن
- **انیمیشن چرخش 3D**: افکت چرخش واقع‌گرایانه برای کارت‌ها
- **رنگ‌بندی زیبا**: گرادیانت‌های مدرن و رنگ‌های چشم‌نواز
- **تجربه کاربری عالی**: بازخورد بصری و صوتی برای تمام اقدامات

### آمار و رکوردها
- **آمار جامع**: تعداد بازی‌ها، بازی‌های تکمیل شده، بهترین زمان و امتیاز
- **جدول رتبه‌بندی**: ثبت ۱۰ رکورد برتر برای هر سطح سختی
- **میانگین حرکات**: محاسبه میانگین حرکات در بازی‌های تکمیل شده
- **مجموع زمان بازی**: ثبت کل زمان صرف شده برای بازی
- **مقایسه عملکرد**: مقایسه رکوردها بر اساس زمان و امتیاز

### ذخیره‌سازی
- **ذخیره خودکار**: تنظیمات، آمار و رکوردها به صورت خودکار ذخیره می‌شوند
- **بدون نیاز به بک‌اند**: تمام داده‌ها در localStorage مرورگر ذخیره می‌شوند
- **پایدار**: اطلاعات پس از بستن مرورگر حفظ می‌شوند

## نحوه استفاده

### شروع بازی
1. روی دکمه "شروع بازی" کلیک کنید
2. کارت‌ها را با کلیک باز کنید و جفت‌های یکسان را پیدا کنید
3. سعی کنید با کمترین حرکت و زمان تمام جفت‌ها را پیدا کنید

### تنظیمات
- **سختی**: انتخاب سطح سختی بازی (آسان، متوسط، سخت)
- **تم کارت‌ها**: انتخاب نوع نمایش روی کارت‌ها (ایموجی، اعداد، رنگ‌ها)
- **افکت صوتی**: فعال/غیرفعال کردن صداها
- **انیمیشن**: فعال/غیرفعال کردن انیمیشن‌ها
- **تایمر**: فعال/غیرفعال کردن زمان‌سنج

### کنترل‌های بازی
- **بازی جدید**: شروع یک بازی جدید
- **توقف/ادامه**: متوقف کردن یا ادامه دادن بازی
- **آمار بازی**: مشاهده آمار کلی
- **جدول رتبه‌بندی**: مشاهده بهترین رکوردها

## ویژگی‌های تکنیکی

### معماری کد
- **الگوی ماژول**: استفاده از IIFE برای ایجاد scope خصوصی
- **تفکیک مسئولیت‌ها**: جداسازی منطق بازی، رابط کاربری و ذخیره‌سازی
- **مدیریت state**: مدیریت حالت بازی با شی‌های مجزا

### عملکرد و بهینه‌سازی
- **الگوریتم shuffle**: استفاده از Fisher-Yates shuffle برای ترکیب تصادفی کارت‌ها
- **Event delegation**: استفاده بهینه از event listener ها
- **Web Audio API**: تولید صداهای ساده بدون نیاز به فایل صوتی
- **CSS animations**: استفاده از transform و transition برای انیمیشن‌های روان
- **3D perspective**: افکت چرخش سه‌بعدی با CSS transforms

### ذخیره‌سازی داده
- **localStorage**: ذخیره تنظیمات و آمار
- **JSON serialization**: تبدیل داده‌ها به فرمت JSON
- **جدول رتبه‌بندی مجزا**: ذخیره رکوردها به تفکیک سطح سختی

## الگوریتم‌ها

### الگوریتم چیدمان کارت‌ها
```
1. انتخاب تعداد جفت بر اساس سطح سختی
2. انتخاب مقادیر از تم انتخابی
3. ساخت آرایه با تکرار هر مقدار (برای ایجاد جفت)
4. shuffle کردن آرایه با الگوریتم Fisher-Yates
5. ایجاد DOM elements برای هر کارت
```

### الگوریتم تطبیق کارت‌ها
```
1. ذخیره کارت اول در حافظه موقت
2. در صورت انتخاب کارت دوم، مقایسه مقادیر
3. اگر یکسان باشند:
   - علامت‌گذاری به عنوان matched
   - محاسبه و اضافه کردن امتیاز
   - بررسی اتمام بازی
4. اگر متفاوت باشند:
   - نمایش انیمیشن shake
   - برگرداندن کارت‌ها به حالت اولیه
5. افزایش شمارنده حرکات
```

### الگوریتم محاسبه امتیاز
```
امتیاز = پاداش_جفت + پاداش_زمان + پاداش_حرکت

پاداش_جفت = 100
پاداش_زمان = max(0, 50 - زمان/2)
پاداش_حرکت = max(0, 30 - تعداد_حرکات)
```

## تم‌های کارت

### ایموجی
مجموعه‌ای از ایموجی‌های مختلف شامل:
- ابزارهای موسیقی
- وسایل ورزشی
- نمادهای بازی

### اعداد
اعداد ۱ تا ۱۸ برای سطوح مختلف

### رنگ‌ها
پالت رنگی شامل:
- رنگ‌های گرم و سرد
- رنگ‌های متنوع و قابل تشخیص
- هارمونی رنگی مناسب

## سطوح سختی

### آسان (4×4)
- 16 کارت (8 جفت)
- مناسب برای مبتدیان
- زمان بازی: 1-3 دقیقه

### متوسط (6×4)
- 24 کارت (12 جفت)
- چالش متعادل
- زمان بازی: 2-5 دقیقه

### سخت (6×6)
- 36 کارت (18 جفت)
- چالش حرفه‌ای
- زمان بازی: 4-8 دقیقه

## نکات بازی

### برای کسب امتیاز بهتر:
1. **تمرکز کنید**: روی مکان کارت‌ها دقت کنید
2. **الگو بسازید**: یک استراتژی منظم برای باز کردن کارت‌ها داشته باشید
3. **سریع عمل کنید**: هر چه سریع‌تر تمام کنید، امتیاز بیشتری می‌گیرید
4. **حرکات کم**: سعی کنید با کمترین حرکت تمام کارت‌ها را پیدا کنید

### برای بهبود حافظه:
- از گوشه‌ها شروع کنید
- کارت‌های باز شده را در ذهن نقشه‌برداری کنید
- در هر حرکت دو کارت جدید را امتحان کنید
- بعد از چند بازی، الگوهای ذهنی بسازید
