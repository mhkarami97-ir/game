## 🎯 درباره پروژه
ابزارها بدون نیاز به سرور و تمام سمت مرورگر خود کاربر اجرا می‌شوند

### 🌟 ویژگی‌های منحصر به فرد:

- ✅ **بدون نیاز به سرور** - تمام عملیات در مرورگر انجام می‌شود
- ✅ **PWA (Progressive Web App)** - قابل نصب روی موبایل و دسکتاپ
- ✅ **دو زبانه** - پشتیبانی کامل از فارسی و انگلیسی
- ✅ **Dark/Light Mode** - تم تاریک و روشن با ذخیره‌سازی
- ✅ **Responsive Design** - طراحی واکنش‌گرا برای تمام دستگاه‌ها
- ✅ **Offline Support** - کار آفلاین با Service Worker
- ✅ **سریع و سبک** - بدون وابستگی‌های سنگین
- ✅ **امن و خصوصی** - داده‌ها روی سرور ذخیره نمی‌شوند

## 🚀 نصب و راه‌اندازی

### پیش‌نیازها

این پروژه نیازی به نصب چیزی ندارد! فقط یک مرورگر مدرن کافی است:
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

### روش 1: استفاده مستقیم از GitHub Pages

1. ریپازیتوری را Fork کنید
2. در تنظیمات، GitHub Pages را فعال کنید
3. آدرس `https://username.github.io/repo-name` را باز کنید

### روش 2: کلون محلی

```bash
# کلون کردن ریپازیتوری
git clone https://github.com/yourusername/game.git

# رفتن به پوشه پروژه
cd game

# باز کردن index.html در مرورگر
# Windows
start index.html

# Mac
open index.html

# Linux
xdg-open index.html
```

### روش 3: استفاده از سرور محلی

```bash
# با Python 3
python -m http.server 8000

# با Node.js (npx)
npx http-server

# با PHP
php -S localhost:8000
```

سپس به `http://localhost:8000` بروید.

## 💡 استفاده

### جستجوی بازی

1. در کادر جستجو تایپ کنید (فارسی یا انگلیسی)
2. یا از کیبورد شورتکات استفاده کنید:
   - `/` - فوکوس به جستجو
   - `Ctrl+K` یا `Cmd+K` - فوکوس به جستجو
   - `Tab` - حرکت بین ابزارها
   - `Enter` - باز کردن ابزار

### تغییر تم

روی دکمه 🌙/☀️ در بالای صفحه کلیک کنید. انتخاب شما ذخیره می‌شود.

### نصب به عنوان اپلیکیشن

#### موبایل (Android)
1. سایت را در Chrome باز کنید
2. روی پیام "نصب اپلیکیشن" در پایین صفحه کلیک کنید
3. یا از منوی Chrome گزینه "Add to Home screen" را انتخاب کنید

#### موبایل (iOS)
1. سایت را در Safari باز کنید
2. روی دکمه Share (📤) کلیک کنید
3. "Add to Home Screen" را انتخاب کنید

#### دسکتاپ
1. در Chrome/Edge آیکون نصب (➕) در نوار آدرس ظاهر می‌شود
2. روی آن کلیک کنید و "Install" را انتخاب کنید

## 📱 PWA و قابلیت نصب

این پروژه یک **Progressive Web App** کامل است با ویژگی‌های زیر:

### ✅ قابلیت‌های PWA

- **📱 نصب روی موبایل**: Android و iOS
- **💻 نصب روی دسکتاپ**: Windows, Mac, Linux
- **🔌 آفلاین**: کار بدون اینترنت با Service Worker
- **⚡ سریع**: کش کردن فایل‌ها برای بارگذاری فوری
- **🔔 اعلان‌ها**: پشتیبانی از Push Notifications (در نسخه‌های بعدی)
- **🎨 Native Feel**: احساس اپلیکیشن بومی

### 📄 فایل‌های PWA

```
manifest.json    # تنظیمات PWA و اطلاعات اپلیکیشن
sw.js           # Service Worker برای کش و آفلاین
favicon.ico     # آیکون اپلیکیشن
```

مستندات کامل PWA در فایل [`PWA-README.md`](./PWA-README.md)

## 🎨 سفارشی‌سازی

### تغییر رنگ‌های تم

در فایل `index.html` متغیرهای CSS را ویرایش کنید:

```css
:root {
    --bg0: #0b1020;          /* پس‌زمینه اصلی */
    --bg1: #0f1730;          /* پس‌زمینه ثانویه */
    --accent: #7c5cff;       /* رنگ اصلی */
    --accent2: #2ee9a6;      /* رنگ ثانویه */
    /* ... */
}
```

### افزودن ابزار جدید

1. یک پوشه جدید بسازید
2. فایل `index.html` ابزار را ایجاد کنید
3. در فایل `index.html` اصلی، ابزار جدید را به آرایه `tools` اضافه کنید:

```javascript
{
    path: 'my-tool',
    faName: 'نام فارسی',
    enName: 'English Name',
    faDesc: 'توضیحات فارسی',
    enDesc: 'English description',
    icon: '🔧'
}
```

## 🤝 مشارکت

مشارکت شما در بهبود این پروژه بسیار ارزشمند است! 🌟

### روش‌های مشارکت

1. **گزارش باگ** - مشکلی پیدا کردید؟ در Issues گزارش دهید
2. **پیشنهاد ویژگی** - ایده‌ای دارید؟ در Discussions مطرح کنید
3. **بهبود کد** - Pull Request بفرستید
4. **افزودن ابزار** - ابزار جدیدی بسازید
5. **بهبود مستندات** - کمک به بهتر شدن README

### مراحل مشارکت

```bash
# 1. Fork کردن پروژه
# 2. کلون کردن fork شما
git clone https://github.com/your-username/game.git

# 3. ساخت branch جدید
git checkout -b feature/my-new-feature

# 4. اعمال تغییرات و commit
git commit -m "Add: my new feature"

# 5. Push کردن
git push origin feature/my-new-feature

# 6. ساخت Pull Request
```

### قوانین Commit

- `Add:` - افزودن ویژگی یا فایل جدید
- `Fix:` - رفع باگ
- `Update:` - به‌روزرسانی کد یا محتوا
- `Remove:` - حذف فایل یا کد
- `Refactor:` - بازنویسی کد
- `Docs:` - تغییر در مستندات
