# بازی سنگ کاغذ قیچی | Rock Paper Scissors Game

یک بازی سنگ کاغذ قیچی کاملاً تحت وب با قابلیت بازی با هوش مصنوعی هوشمند

## ویژگی‌های اصلی

### قابلیت‌های بازی
- **چند حالت بازی**: نامحدود، بهترین از ۳، بهترین از ۵، بهترین از ۷
- **هوش مصنوعی پیشرفته**: سه سطح سختی (آسان، متوسط، سخت)
  - **آسان**: انتخاب تصادفی ساده
  - **متوسط**: ۳۰٪ احتمال پیش‌بینی حرکت بعدی بازیکن
  - **سخت**: تحلیل الگوی حرکات بازیکن و پیش‌بینی هوشمند
- **انیمیشن‌های روان**: انیمیشن‌های زیبا برای نمایش حرکات و نتایج
- **بازخورد بصری**: رنگ‌های متفاوت برای برد، باخت و مساوی
- **سیستم امتیازدهی**: ثبت امتیاز بازیکن و کامپیوتر در هر دور

### پشتیبانی چند زبانه
- **فارسی**: به‌عنوان زبان پیش‌فرض با طراحی RTL
- **انگلیسی**: با طراحی LTR
- تبدیل خودکار جهت صفحه با تغییر زبان
- ترجمه کامل تمام المان‌های رابط کاربری
- استفاده از ایموجی‌های جهانی برای نمایش سنگ، کاغذ و قیچی

### طراحی و رابط کاربری
- **طراحی ریسپانسیو**: سازگار با تمام اندازه‌های صفحه (دسکتاپ، تبلت، موبایل)
- **حالت تاریک/روشن**: تم‌های قابل تعویض با طراحی مدرن
- **طراحی زیبا**: استفاده از رنگ‌بندی حرفه‌ای و سایه‌های لطیف
- **تجربه کاربری عالی**: انیمیشن‌های روان و بازخورد بصری واضح
- **نمایش بصری انتخاب‌ها**: ایموجی‌های بزرگ و رنگ‌بندی مناسب برای هر نتیجه

### آمار و تاریخچه
- **ثبت آمار بازی**: تعداد کل بازی‌ها، تعداد برد، باخت و مساوی
- **نرخ برد**: محاسبه و نمایش درصد برد بازیکن
- **تاریخچه بازی‌ها**: ذخیره ۱۰ بازی آخر با جزئیات کامل
  - نتیجه بازی (برد/باخت)
  - امتیاز نهایی
  - تاریخ و زمان بازی
  - تعداد دورهای بازی شده
- **نمایش بصری تاریخچه**: رنگ‌بندی متفاوت برای برد و باخت
- **بازنشانی آمار**: امکان پاک کردن تمام آمار با تأیید کاربر

### ذخیره‌سازی
- **ذخیره خودکار**: تنظیمات و آمار به صورت خودکار ذخیره می‌شوند
- **بدون نیاز به بک‌اند**: تمام داده‌ها در localStorage مرورگر ذخیره می‌شوند
- **پایدار**: اطلاعات پس از بستن مرورگر حفظ می‌شوند

## نحوه استفاده

### راه‌اندازی
1. فایل `index.html` را در مرورگر باز کنید
2. روی دکمه "شروع بازی" کلیک کنید
3. یکی از گزینه‌های سنگ، کاغذ یا قیچی را انتخاب کنید
4. از بازی لذت ببرید!

### قوانین بازی
- **سنگ** قیچی را می‌شکند
- **قیچی** کاغذ را می‌برد
- **کاغذ** سنگ را می‌پوشاند
- در صورت انتخاب یکسان، نتیجه مساوی است

### حالت‌های بازی
- **نامحدود**: بازی تا زمانی که خودتان بخواهید ادامه دارد
- **بهترین از ۳**: اولین کسی که ۲ امتیاز بگیرد برنده است
- **بهترین از ۵**: اولین کسی که ۳ امتیاز بگیرد برنده است
- **بهترین از ۷**: اولین کسی که ۴ امتیاز بگیرد برنده است

### سطوح سختی
- **آسان**: کامپیوتر به صورت تصادفی انتخاب می‌کند
- **متوسط**: کامپیوتر گاهی اوقات حرکت شما را پیش‌بینی می‌کند
- **سخت**: کامپیوتر الگوی حرکات شما را تحلیل کرده و سعی در پیروزی دارد

## تنظیمات

### حالت بازی
می‌توانید حالت بازی را از بخش تنظیمات تغییر دهید. این تنظیمات شامل:
- نامحدود (بدون محدودیت امتیاز)
- بهترین از ۳، ۵ یا ۷

### سطح سختی
سطح هوش مصنوعی را انتخاب کنید:
- آسان: برای تازه‌کاران
- متوسط: برای بازی‌های معمولی
- سخت: برای چالش‌های واقعی

### انیمیشن و صدا
می‌توانید انیمیشن‌ها و صداها را فعال یا غیرفعال کنید.

## ویژگی‌های فنی

### معماری کد
- **الگوی Module**: استفاده از IIFE برای ایجاد namespace خصوصی
- **تفکیک مسئولیت**: جداسازی منطق بازی، UI و مدیریت داده
- **کد تمیز**: استفاده از نام‌گذاری معنادار و ساختار واضح
- **OOP**: استفاده از اصول برنامه‌نویسی شی‌گرا

### الگوریتم هوش مصنوعی
- **تحلیل الگو**: بررسی سه حرکت آخر بازیکن
- **پیش‌بینی هوشمند**: استفاده از داده‌های تاریخی برای پیش‌بینی
- **تصادفی‌سازی**: ترکیب استراتژی با تصادفی‌سازی برای طبیعی‌تر شدن بازی

### مدیریت حالت
- **State Management**: مدیریت متمرکز state بازی
- **LocalStorage**: ذخیره‌سازی پایدار تنظیمات و آمار
- **Real-time Updates**: به‌روزرسانی فوری رابط کاربری

### بهینه‌سازی
- **عملکرد بالا**: استفاده از Event Delegation و کش کردن المان‌ها
- **حافظه کارآمد**: مدیریت بهینه داده‌های تاریخچه (محدود به ۱۰ بازی)
- **CSS مدولار**: استفاده از متغیرهای CSS برای تم‌بندی آسان

- **Success (سبز)**: برای نمایش برد
- **Danger (قرمز)**: برای نمایش باخت

### فاصله‌گذاری و چیدمان

## راهنمای توسعه

### افزودن ویژگی جدید
1. تابع مربوطه را در ماژول `RockPaperScissorsGame` اضافه کنید
2. رابط کاربری لازم را در HTML ایجاد کنید
3. استایل‌های مورد نیاز را در CSS بنویسید
4. ترجمه‌های لازم را در `translations.json` اضافه کنید

### تغییر طراحی
1. متغیرهای CSS را در `:root` ویرایش کنید
2. برای تم تاریک، متغیرها را در `[data-theme="dark"]` تغییر دهید
3. از متغیرها در کل CSS استفاده کنید تا تغییرات یکپارچه باشد
