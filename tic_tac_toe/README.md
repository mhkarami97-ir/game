# بازی دوز | Tic-Tac-Toe Game

یک بازی دوز (XO) کاملاً تحت وب با قابلیت بازی دو نفره یا بازی با هوش مصنوعی

## ویژگی‌های اصلی

### قابلیت‌های بازی
- **دو حالت بازی**: بازی دو نفره یا بازی با کامپیوتر
- **هوش مصنوعی پیشرفته**: سه سطح سختی (آسان، متوسط، سخت) با الگوریتم Minimax
- **راهنمایی هوشمند**: نمایش بهترین حرکت ممکن برای بازیکن
- **امکان برگشت**: بازگشت به حرکت قبلی
- **انیمیشن‌های روان**: انیمیشن‌های زیبا برای حرکات و برد
- **افکت‌های صوتی**: صداهای مختلف برای حرکات، برد و مساوی

### پشتیبانی چند زبانه
- **فارسی**: به‌عنوان زبان پیش‌فرض با طراحی RTL
- **انگلیسی**: با طراحی LTR
- تبدیل خودکار جهت صفحه با تغییر زبان
- ترجمه کامل تمام المان‌های رابط کاربری

### طراحی و رابط کاربری
- **طراحی ریسپانسیو**: سازگار با تمام اندازه‌های صفحه (دسکتاپ، تبلت، موبایل)
- **حالت تاریک/روشن**: تم‌های قابل تعویض با طراحی مدرن
- **طراحی زیبا**: استفاده از رنگ‌بندی حرفه‌ای و سایه‌های لطیف
- **تجربه کاربری عالی**: انیمیشن‌های روان و بازخورد بصری مناسب

### آمار و تاریخچه
- **ثبت آمار بازی**: تعداد کل بازی‌ها، تعداد برد هر بازیکن، تعداد مساوی
- **تاریخچه بازی‌ها**: ذخیره ۱۰ بازی آخر با جزئیات (برنده، تاریخ، زمان، تعداد حرکات)
- **نمایش نمودارها**: نمایش آمار به صورت بصری و جذاب
- **بازنشانی آمار**: امکان پاک کردن تمام آمار

### ذخیره‌سازی
- **ذخیره خودکار**: تنظیمات، آمار و تاریخچه به صورت خودکار ذخیره می‌شوند
- **بدون نیاز به بک‌اند**: تمام داده‌ها در localStorage مرورگر ذخیره می‌شوند
- **پایدار**: اطلاعات پس از بستن مرورگر حفظ می‌شوند

## راهنمای بازی

### شروع بازی
1. روی دکمه "شروع بازی" کلیک کنید
2. حالت بازی را انتخاب کنید (دو نفره یا با کامپیوتر)
3. در صورت انتخاب بازی با کامپیوتر، سطح سختی را تعیین کنید
4. روی خانه‌های خالی کلیک کنید تا علامت خود را قرار دهید

### قوانین بازی
- بازیکن X همیشه شروع می‌کند
- بازیکنان به نوبت علامت خود (X یا O) را در خانه‌های خالی قرار می‌دهند
- اولین بازیکنی که ۳ علامت در یک خط (افقی، عمودی یا مورب) قرار دهد برنده است
- اگر تمام خانه‌ها پر شوند و هیچ برنده‌ای نباشد، بازی مساوی است

### دکمه‌های کنترل
- **بازی جدید**: شروع یک بازی جدید
- **برگشت**: بازگشت به حرکت قبلی (در حالت دو نفره یک حرکت، در حالت با کامپیوتر دو حرکت)
- **راهنمایی**: نمایش بهترین حرکت ممکن
- **آمار بازی**: مشاهده آمار و تاریخچه بازی‌ها

### تنظیمات
- **حالت بازی**: دو نفره یا با کامپیوتر
- **سختی**: آسان (تصادفی)، متوسط (ترکیبی)، سخت (الگوریتم Minimax)
- **افکت صوتی**: فعال/غیرفعال کردن صداها
- **انیمیشن**: فعال/غیرفعال کردن انیمیشن‌ها

## ویژگی‌های فنی

### تکنولوژی‌های استفاده شده
- **HTML5**: ساختار معنایی و استاندارد
- **CSS3**: استایل‌دهی مدرن با متغیرهای CSS و انیمیشن‌ها
- **JavaScript (ES5+)**: منطق بازی با استفاده از الگوهای طراحی

### معماری کد
- **الگوی Module Pattern**: کپسوله‌سازی کد و جلوگیری از آلودگی فضای نام جهانی
- **تفکیک نگرانی‌ها**: جداسازی منطق بازی، رابط کاربری و مدیریت داده
- **استفاده از var**: مطابق با استانداردهای ES5 برای سازگاری بیشتر
- **کد تمیز**: نام‌گذاری واضح و ساختار منظم

### الگوریتم هوش مصنوعی
- **Minimax Algorithm**: الگوریتم بهینه برای بازی‌های دو نفره
- **Depth Limitation**: بهینه‌سازی عملکرد با محدودیت عمق جستجو
- **Scoring System**: سیستم امتیازدهی برای انتخاب بهترین حرکت

### ذخیره‌سازی داده
- **localStorage**: ذخیره تنظیمات، آمار و تاریخچه
- **JSON**: فرمت استاندارد برای ذخیره و بازیابی داده‌ها
- **Error Handling**: مدیریت خطاها در خواندن و نوشتن داده‌ها

## بهینه‌سازی‌ها

### عملکرد
- کد بهینه شده برای اجرای سریع
